# TypeScript standards

Standards for TypeScript / JavaScript code in the Pyth Crosschain monorepo. The
cross-cutting [style.md](../style.md), [testing.md](../testing.md), and
[services.md](../services.md) also apply.

- [style.md](style.md) — Biome, `catalog:` dependency versions, and layered
  architecture.
- [packages.md](packages.md) — creating a new package with
  `pnpm create-pyth-package`, and package best practices.
- [testing.md](testing.md) — test-driven development for TypeScript.
