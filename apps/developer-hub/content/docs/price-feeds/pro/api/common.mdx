## Authentication

All Pyth Pro APIs (except Proxy) require authentication with an access token. See [Acquire Access Token](/price-feeds/pro/acquire-access-token) for instructions.

**Format**: Bearer token in `Authorization` header

```
Authorization: Bearer YOUR_ACCESS_TOKEN
```

<Callout type="warning">
  **Security**: Never expose access tokens in frontend code or client-side applications. Access tokens should only be used in secure backend environments.
</Callout>

## Rate Limits

### WebSocket

| Limit | Value | Scope |
|-------|-------|-------|
| Connections per token | 10 | Per access token |
| Subscriptions per connection | 100 | Per WebSocket connection |
| Price feeds per subscription | 100 | Per subscribe message |
| Subscribe messages per minute | 60 | Per connection |

### REST API

| Endpoint | Limit | Window |
|----------|-------|--------|
| `GET /v1/latest_price` | 100 requests | Per second |

**Rate Limit Headers** (included in responses):

| Header | Description |
|--------|-------------|
| `X-RateLimit-Limit` | Maximum requests per window |
| `X-RateLimit-Remaining` | Remaining requests in current window |
| `X-RateLimit-Reset` | Unix timestamp when window resets |

### History Service

| Endpoint | Limit | Window |
|----------|-------|--------|
| `GET /history/v1/symbols` | 100 requests | Per minute |
| `GET /history/v1/ohlc` | 60 requests | Per minute |
| `GET /history/v1/prices` | 60 requests | Per minute |

## Error Codes

### WebSocket Application Errors

| Code | Description | Resolution |
|------|-------------|------------|
| `UNAUTHORIZED` | Invalid or missing access token | Verify token, obtain new token if expired |
| `INVALID_SUBSCRIPTION` | Malformed subscription message | Check message format and parameters |
| `INVALID_PRICE_FEED` | Unknown price feed ID | Check [Price Feed IDs](/price-feeds/pro/price-feed-ids) |
| `INVALID_PROPERTY` | Unknown property requested | Check available properties |
| `INVALID_CHANNEL` | Unknown channel type | Use valid channel: `real_time`, `fixed_rate@Xms` |
| `RATE_LIMITED` | Too many requests | Reduce request frequency, implement backoff |
| `SUBSCRIPTION_LIMIT` | Maximum subscriptions reached | Unsubscribe from unused feeds |
| `INTERNAL_ERROR` | Server-side error | Retry with exponential backoff |

### WebSocket Close Codes

| Code | Name | Meaning | Action |
|------|------|---------|--------|
| `1000` | Normal Closure | Clean disconnect | None - normal behavior |
| `1001` | Going Away | Server shutting down | Reconnect to another endpoint |
| `1006` | Abnormal Closure | Connection lost unexpectedly | Reconnect with backoff |
| `1008` | Policy Violation | Authentication failed | Check credentials |
| `1011` | Internal Error | Unexpected server error | Reconnect with backoff |
| `4001` | Unauthorized | Access token invalid | Refresh token, then reconnect |
| `4003` | Forbidden | Insufficient permissions | Check token permissions |
| `4029` | Rate Limited | Too many connection attempts | Wait before reconnecting |

### REST API HTTP Status Codes

| Status | Code | Description | Common Causes |
|--------|------|-------------|---------------|
| `400` | `INVALID_REQUEST` | Malformed request | Missing required parameters, invalid format |
| `401` | `UNAUTHORIZED` | Authentication failed | Missing, expired, or invalid token |
| `403` | `FORBIDDEN` | Access denied | Token lacks required permissions |
| `404` | `NOT_FOUND` | Resource not found | Invalid endpoint or price feed ID |
| `405` | `METHOD_NOT_ALLOWED` | Invalid HTTP method | Using POST instead of GET |
| `429` | `RATE_LIMITED` | Rate limit exceeded | Too many requests in time window |
| `500` | `INTERNAL_ERROR` | Server error | Temporary server issue |
| `502` | `BAD_GATEWAY` | Upstream error | Backend service unavailable |
| `503` | `SERVICE_UNAVAILABLE` | Service down | Temporary maintenance |
| `504` | `GATEWAY_TIMEOUT` | Request timeout | Backend taking too long |

**Error Response Format**:

```json
{
  "error": "ERROR_CODE",
  "message": "Human-readable description of the error"
}
```

### History Service Error Codes

| Status | Code | Description |
|--------|------|-------------|
| `400` | `INVALID_SYMBOL` | Symbol format is invalid |
| `400` | `INVALID_RESOLUTION` | Resolution not supported |
| `400` | `INVALID_TIME_RANGE` | From/to timestamps invalid |
| `404` | `SYMBOL_NOT_FOUND` | Requested symbol doesn't exist |
| `429` | `RATE_LIMITED` | Too many requests |
| `500` | `INTERNAL_ERROR` | Server error |
