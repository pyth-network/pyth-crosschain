## Services

| Service                                     | Base URL                                           | Use Case                                  |
| ------------------------------------------- | -------------------------------------------------- | ----------------------------------------- |
| [WebSocket](/price-feeds/pro/api/websocket) | `wss://pyth-lazer-{0,1,2}.dourolabs.app/v1/stream` | Real-time streaming price updates         |
| [REST](/price-feeds/pro/api/rest)           | `https://pyth-lazer.dourolabs.app`                 | On-demand price queries                   |
| [History](/price-feeds/pro/api/history)     | `https://history.pyth-lazer.dourolabs.app`         | Historical data and OHLC candlestick data |
| [Proxy (Beta)](/price-feeds/pro/api/proxy)  | `https://pyth-lazer-proxy-3.dourolabs.app`         | Simplified access without authentication  |
