## Choosing the Right API

| Use Case | Recommended API | Why |
|----------|-----------------|-----|
| **Real-time trading** | WebSocket | Lowest latency, continuous updates |
| **Price displays/dashboards** | WebSocket | Live updates without polling |
| **Periodic price checks** | REST | Simple request/response |
| **Bootstrap initial state** | REST | Single request for current prices |
| **Charting applications** | History | OHLC data for candlestick charts |
| **Backtesting** | History | Historical price data |
| **Analytics** | History | Time-series data analysis |

## Base URLs

| Service | Base URL |
|---------|----------|
| WebSocket Streaming | `wss://pyth-lazer-{0,1,2}.dourolabs.app/v1/stream` |
| REST API | `https://pyth-lazer.dourolabs.app` |
| History Service | `https://history.pyth-lazer.dourolabs.app` |

## Authentication

All Pyth Pro APIs require authentication with an access token. See [Acquire Access Token](/price-feeds/pro/acquire-access-token) for instructions.

```typescript
// WebSocket (via SDK)
const client = await PythLazerClient.create(endpoints, "YOUR_TOKEN");

// REST API
const response = await fetch(url, {
  headers: { Authorization: `Bearer YOUR_TOKEN` },
});
```

<Callout type="warning">
  **Security**: Never expose access tokens in frontend code or client-side applications.
</Callout>

## Quick Start

### Real-time Prices (WebSocket)

```typescript
import { PythLazerClient } from "@pythnetwork/pyth-lazer-sdk";

const client = await PythLazerClient.create(
  [
    "wss://pyth-lazer-0.dourolabs.app/v1/stream",
    "wss://pyth-lazer-1.dourolabs.app/v1/stream",
    "wss://pyth-lazer-2.dourolabs.app/v1/stream",
  ],
  process.env.ACCESS_TOKEN!
);

client.subscribe({
  type: "subscribe",
  subscriptionId: 1,
  priceFeedIds: [1, 2], // BTC, ETH
  properties: ["price"],
  channel: "fixed_rate@200ms",
});

client.addMessageListener((message) => {
  console.log(message);
});
```

### Current Price (REST)

```bash
curl "https://pyth-lazer.dourolabs.app/v1/latest_price?price_feed_ids=1,2" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### Historical Data (History)

```bash
curl "https://history.pyth-lazer.dourolabs.app/history/v1/ohlc?symbol=BTC/USD&resolution=60&from=1704067200&to=1704153600"
```
