<Callout type="info">
  You can also access the list of symbols programmatically via the [Symbols List
  API](https://history.pyth-lazer.dourolabs.app/history/v1/symbols).

You can optionally filter results using:

- `query`: filter by symbol name (e.g., [?query=AAPL](https://history.pyth-lazer.dourolabs.app/history/v1/symbols?query=AAPL))
- `asset_type`: filter by asset type (e.g., [?asset_type=crypto](https://history.pyth-lazer.dourolabs.app/history/v1/symbols?asset_type=crypto))

You can use either parameter, both, or neither.
